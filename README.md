# XXiOSProject
## 一、项目架构
1. `AppDelegate`   相关配置
2. `Module`    开发模块
3. `Manager`    全局管理—\>单例、用户信息等
4. `Resource`    资源
5. `Vendor`    第三方
6. `Utils`   工具
	 * `NetWork`    网络请求
	* `Global`    全局
	* `BaseClass`    基类
	* `Category`    分类
	* `Controls`    自定义控件
	* `DataBase`    数据库
